Aquí encontrarás algunas posibles soluciones si presentas problemas en la página.

No funciona el iframe de YouTube

El iframe que muestra el último video subido por ElmichiYT puede tener problemas por las siguientes posibles razones.

• Bloqueador de anuncios: Desactiva el bloqueador de anuncios y revisa si funciona el iframe. Si ese es el caso, tu bloqueador de anuncios es el problema.

• API sin créditos: el sitio web usa una API gratuita de Google que muestra el último video de ElmichiYT. Si esa API se queda sin créditos, ya no funcionará hasta dentro de 24h.

• Mala conexión a Internet: Sí, los embebidos son pesaditos. Si tu internet es muy lento, podría tardar más de lo normal. Pero esto probablemente no sea.

• Caída de la plataforma: Aunque poco probable, probablemente YouTube (o la API) esté caído. La solución es esperar

Página de Error 404

¿Te apareció esta página en alguna URL? No te preocupes. Hay 3 razones por las que te puede aparecer.

• URL mal escrita: GitHub Pages es muy especialito. Si en una URL escribes algo como "/Games/" en vez de "/games", te lanzará dicho error.

• Página removida: Es probable que dicha página o archivo al que deseaba entrar, haya sido removido.

• Error interno: Posiblemente haya un error en la página. Un error hecho por mí pero pronto será solucionado. (si me doy cuenta)

La solución es simplemente ir a otra parte de la página funcional, o esperar a que se elimine el error si la página existe.
