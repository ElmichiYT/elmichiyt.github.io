Símbolo del sistema

Consola procesadora de comandos de Windows


¿Qué es?

El Símbolo del sistema de Windows es una terminal para hacer acciones mediante CLI.

Puedes ver más información aquí: Símbolo del sistema de Windows - Wikipedia


¿Mi Windows lo tiene?

Windows NT 3.1 y posteriores incluían cmd.exe instalado, hasta que en Windows XP fue el procesador de comandos predeterminado.

Si no tienes una computadora de 1990, SÍ O SÍ tienes cmd.exe instalado en tu PC. Aunque hay un problema: la funcionalidad del Símbolo del sistema depende de la versión del sistema operativo. En pocas palabras: en cmd.exe de Windows 10 puede funcionar cURL, pero en Windows XP no. Puede que no esté instalado de fábrica se tenga que instalar manualmente o el programa no exista para dicho sistema.
